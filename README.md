# Student Management System (Java)

## Description
This is a simple Student Management System developed using Core Java, JDBC, and MySQL.
It allows users to perform basic CRUD operations on student records.

## Technologies Used
- Java
- JDBC
- MySQL
- OOP Concepts

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## How to Run
1. Create database in MySQL
2. Update database credentials in Java file
3. Run the main Java class
